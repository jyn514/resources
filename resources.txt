Linux:

    Tools:
bash: A 'shell'; takes input and executes programs. Has it's own syntax.
ssh: 'Secure shell'; connects to a remote computer and executes programs. www.openssh.com/
make: Automated build tool (so you don't have to type as much). www.gnu.org/software/make/
apt: package manager (similar to an app store). wiki.debian.org/Apt
coreutils: most of the tools we will use in this course. www.gnu.org/software/coreutils/manual/html_node/index.html
gcc: multi-language compiler. In this course we will be using g++, but all of the gnu compilers are often used. gcc.gnu.org/onlinedocs/gcc-7.2.0/gcc/
binutils: tools necessary for compiling programs (assemblers, hexdumps, etc). www.gnu.org/software/binutils/
git: version control, so you can keep track of the changes you've made. https://git-scm.com/book/en/v2, https://reddit.com/r/git
github: free online hosting for your git repos. https://github.com/
editors: vi/vim, emacs, nano, pico, etc. etc.
other useful tools: which, type, help, info, man, find/locate, umask, zip, tar, gz/bzip2, wget/curl/telnet, passwd, less/more/pg, screen, cron/crond/crontab/anacron, gnupg, ip/ipconfig/dhclient, anything in coreutils or `help`.
