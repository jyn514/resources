Oracle Database XE installation ... Though the installation process is quite simple and streamlined, there are a few tips that will help ensure smoother installation and operation of your Oracle Database XE server.

Pre-installation Tasks (will these be the same for XE?)

  • Create an operating system group called dba.
  • Create an operating system user account (such as oracle) that will be used to install the software.
  • Assign the oracle user account to the dba group.
  • Check the kernel parameters to ensure sufficient memory allocation settings. See the product documentation for recommended values.
  • Verify that ORACLE_HOME environment variable is not set.
